📚 Bank Management Application (C++)

A console-based Bank Management Application developed using C++ that simulates core banking operations. The system uses object-oriented programming concepts and file handling to ensure persistent and secure customer data storage.

🚀 Features

➕ Deposit funds into accounts
➖ Withdraw funds from accounts
📋 Check account balances
💾 Store customer data securely using file management
📑 Menu-driven interface for easy navigation

🛠️ Technologies Used

C++ (Object-Oriented Programming)
File Handling (for secure data storage)

📂 Project Structure

bank-management-system-cpp/
│
├── main.cpp            # Main source code
├── bank_records.dat    # File used for storing customer data
├── README.md           # Project documentation
▶️ How to Run

Download or clone the repository
Open the project folder
Compile the code using a C++ compiler
Run the executable file

🧠 Concepts Used

Object-Oriented Programming (Classes and Objects)
File handling (read/write/update operations)
Conditional statements and loops
Menu-driven program design

📌 Note

The program uses bank_records.dat file to store and retrieve customer records.
Ensure the file is present in the same directory as the source code while running the program or that the program has permissions to create it.
