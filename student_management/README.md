# 📚 Student Management System (C++)

A console-based Student Management System developed using C++ that allows efficient management of student records through a menu-driven interface. The system uses file handling to ensure persistent data storage.

---

## 🚀 Features

- ➕ Add new student records  
- 📋 Display all student details  
- ✏️ Update existing student information  
- ❌ Delete student records  
- 💾 Store data persistently using file handling  
- 📑 Menu-driven interface for easy navigation  

---

## 🛠️ Technologies Used

- C++ (Core Programming)
- File Handling (for data storage)

---
